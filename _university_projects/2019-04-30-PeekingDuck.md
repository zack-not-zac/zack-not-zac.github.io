CMP311 - An Android application written in Java to execute, queue, and edit RubberDucky scripts.

<div style="overflow: auto">
	<img src="{{ site.url }}/images/PeekingDuck.png" width="66%" height="66%" style="float: left" />

	<p>The application was made for use with Kali NetHunter, however, should work with any phone with a modified kernel to allow it to be recognised as a HiD-Compliant keyboard by a victim computer. A list of these devices can be found <a href="https://github.com/pelya/android-keyboard-gadget">here</a>.<br /><br />

	This application was developed as part of a group project for the CMP311 module.<br /><br />

	Source code is available <a href="https://github.com/zack-not-zac/PeekingDuck">here</a>.
